<Update id="v1-3-0" label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- **Pattern Learning**: ForgeKit now analyzes your completion times and adapts task suggestions to your most productive periods.
- **Smart Scheduling**: Automatic daily task generation based on room priorities and your past behavior.

## Improvements

- Calming design refinements with softer gradients for reduced visual fatigue.
- Offline mode enhancements for instant task completion feedback.

## Bug Fixes

- Fixed occasional sync issues with streak counters.
- Resolved notification delays on certain devices.

</Update>

<Update id="v1-2-0" label="2024-09-20" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- **Gentle Reminders**: Customizable, supportive notification tones and messages.
- **Privacy Dashboard**: View and manage all local data insights directly in the app.

## Improvements

- Faster task loading with optimized offline-first architecture.
- Better accessibility support for voice-over users.

## Bug Fixes

- Corrected task duplication in daily lists.
- Fixed streak reset on app backgrounding.

</Update>

<Update id="v1-1-0" label="2024-08-10" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- **Celebration Animations**: Dopamine-boosting effects on task completion.
- **Streak Sharing**: Optional social sharing of your progress.

## Breaking Changes

- Updated task format requires re-onboarding for existing users.

## Bug Fixes

- Patched initial load crash on low-memory devices.
- Improved battery usage during background syncs.

</Update>

## How to Update

Follow these steps to keep ForgeKit running smoothly.

<Steps>
  <Step title="Check for Updates" icon="download">
    Open the Google Play Store (or App Store when available).
    
    Search for `ForgeKit` or `Tidy`.
  </Step>
  <Step title="Install" icon="check-circle">
    Tap **Update** if available.
    
    Wait for the download to complete.
  </Step>
  <Step title="Restart App" icon="refresh-cw">
    Force close and reopen ForgeKit.
    
    Review the new changelog in-app.
  </Step>
</Steps>
