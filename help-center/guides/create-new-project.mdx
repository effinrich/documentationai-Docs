title: Creating a New Project
description: Start a new documentation project in ForgeKit.
---

## Overview

Projects help you group related docs.

## Steps

<Steps>
  <Step title="Open dashboard">
    Click "Projects" in the main menu.
  </Step>
  <Step title="Click New Project">
    Press the "New Project" button.
  </Step>
  <Step title="Set project name">
    Enter a name, e.g., "Acme API Docs".
  </Step>
  <Step title="Finish setup">
    Click "Create". Your project appears in the list.
  </Step>
</Steps>
