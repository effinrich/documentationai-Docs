---
title: Can't Save Changes
description: Fix issues saving documentation edits.
---

## Symptoms

- The "Save" button is disabled
- Changes aren't saved

## Causes

- No write access
- Session expired

## Solution

<Steps>
  <Step title="Check permissions">
    Make sure you have editing rights for the project.
  </Step>
  <Step title="Refresh session">
    Log out and log in again to refresh authentication.
  </Step>
  <Step title="Retry save">
    Try editing and saving again after refresh.
  </Step>
</Steps>
